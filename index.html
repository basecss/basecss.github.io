使用Octopress在Github搭建博客

Sep 19th, 2013

Octopress折腾笔记。

安装ruby依赖

Mac下ruby的默认版本是1.8.7，新版的Octopress需要1.9.3及以上版本的ruby支持，先升级下ruby版本。

各种搜索的结果之下，最终决定使用rvm(ruby多版本控制器)来升级ruby。由于之前已经通过XCode安装了Command line tools，下面直接先安装好rvm：

curl -L get.rvm.io | bash -s stable

rvm install ruby 1.9.3

安装Octopress

直接将Octopress程序clone到本地安装即可。

git clone https://github.com/imathis/octopress.git dist

注意这里的dist可以自行的指定目录名。比如我在本地指定的是basecss[在我的Documents的Github目录中]。

接下来安装相关依赖：

gem install bundler

bundle install

最后在本地安装Octopress：

rake install

配置Octopress

clone到本地的文件中有一个_config.yml文件，修改其中的相关配置即可。

新增博客文章

rake new_post['fileName']

这回在本地的source/_post中生成响应的markdown文件，使用markdown写好文章即可。

发布博客

可以使用下面的命令在本地预览一下：

rake preview

托管到github

rake setup_github_pages

根据提示配置好即可。(注意国内最好用http协议的git url)

rake generate
rake deploy

完成发布
